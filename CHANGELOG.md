# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-03-28

### Added

- Initial VS Code extension scaffold.
- Telegram bot integration with polling mode.
- Codex session watcher for local session files.
- `/tasks` list in Telegram for recent sessions.
- Session selection with latest 5-message preview.
- Independent Telegram chat mode (separate from VS Code Codex thread).
- Configurable `codexbridge.codexBinary`.
- Basic publishing metadata and MIT license.
